# Design References

Ця папка призначена для референсів проєкту `Surfing_template`.

Сюди можна додавати:

- скриншоти оригінального дизайну;
- desktop/mobile QA screenshots;
- фото-референси хвиль, серферів, пляжів, курсів і локацій;
- проміжні візуальні порівняння;
- нотатки до зображень або композицій.

Рекомендовані назви:

```text
hero-reference-desktop.png
courses-card-reference.png
qa-home-desktop-1440.png
qa-home-mobile-390.png
```

Не зберігай тут секрети, приватні документи або файли з незрозумілою ліцензією.
