# Surfing Institute

## Опис

Інститут серфінгу пропонує уроки серфінгу для початківців, коучинг для просунутих, дитячі серфінг-табори, навчання з безпеки на пляжі та заняття з серфінгу з гідом у місцях для серфінгу світового класу.

URL: [https://surfing-template.vercel.app/](https://surfing-template.vercel.app/)
